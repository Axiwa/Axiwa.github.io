Scala notes

Elements Evaluation strategy Call-by-value: A strict evaluation strategy, in which the expression is evaluated and bound to the corresponding parameter before the function body is evaluated. The val form is CBV. val z ={ println("z"); 3 } Running this will give z val z: Int = 3 Calling z will not print “z” because it has been evaluated while defined. Scala uses CBV by default. Call-by-name: A non-strict evaluation strategy which will defer the evaluation of the expression until the program needs it.
